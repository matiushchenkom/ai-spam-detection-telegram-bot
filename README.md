# AI Spam Detection Telegram Bot

This project is an AI-based spam and phishing message detection system integrated with a Telegram bot.  
The main goal of the project is to detect suspicious text messages and classify them as either safe or spam/phishing.

The system uses natural language processing techniques and a neural network model trained on spam message datasets.

## Features

- Text preprocessing using NLTK
- Spam and phishing message classification
- TensorFlow/Keras neural network model
- Telegram bot integration
- Message prediction with confidence percentage
- Model evaluation using classification report and confusion matrix
- Training and validation visualization

## Technologies Used

- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib
- python-telegram-bot
- python-dotenv

## Dataset

The model was trained using spam message datasets, including:
- SMS Spam Collection Dataset
